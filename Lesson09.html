Using a Number Line

Lesson Overview

ST Math Game:

In the game, JiJi Cycle Select Basket, players need to get JiJi on the balloon basket. Place the basket where JiJi will land and JiJi gets to fly up. If the basket is not near enough to where JiJi lands, the balloons go up without JiJi and players need to try again. The JiJi cycle rolls to the right for the blue wheels and fraction of a wheel and to the left for red wheels or fraction of a wheel. The game connects a circle model of fractions to a linear, number line model. The game also builds a visual understanding of adding and subtracting fractions in an estimation context.

Students will:

Build a number line to model an addition problem with unlike denominators. Students will apply their understanding of partitioning a number line so that tick marks can be used to plot equivalent fractions with like denominators (e.g., partition a number line to plot thirds and sixths). Model addition of fractions as directional jumps on a number line.

Teachers will:

Task 1: Present ST Math puzzle

Project a puzzle from level 2, unit fractions of fourths and thirds. Students analyze the animation and notice how the fraction circles roll out onto the number line. Ask students to build a number line and show where each fraction piece lands. Students represent a puzzle and its solution with equations (addition and multiplication). Project a puzzle from level 3. Have students represent the animation on a number line as above. Discuss how the fraction pieces can be decomposed into unit fractions with like denominators.

Task 2: Pose the problem

Draw a number line and locate 5⁄ 4  on your number line. Locate the point on your number line 4⁄ 3  to the right of 5⁄ 4. (possible numbers: 3⁄ 8  + 7⁄ 8; 2⁄ 3  + 3⁄ 6; 2⁄ 3  + 3⁄ 4)

Instructional Goals:

Common Core Content Standards

Extend understanding of fraction equivalence and ordering.

4.NF.A.1 Explain why a fraction a⁄ b  is equivalent to a fraction (n × a)⁄ (n × b)  by using visual fraction models, with attention to how the number and size of the parts differ even though the two fractions themselves are the same size. Use this principle to recognize and generate equivalent fractions.

Build fractions from unit fractions by applying and extending previous understandings of operations on whole numbers.

4.NF.B.3 Understand a fraction a⁄ b  with a> 1 as a sum of fractions 1⁄ b.

4.NF.B.3.a Understand addition and subtraction of fractions as joining and separating parts referring to the same whole.

Use equivalent fractions as a strategy to add and subtract fractions.

5.NF.1 Add and subtract fractions with unlike denominators (including mixed numbers) by replacing given fractions with equivalent fractions in such a way as to produce an equivalent sum or difference of fractions with like denominators. For example, 2⁄ 3  + 5⁄ 4  = 8⁄ 12  + 15⁄ 12  = 23⁄ 12. (In general, a⁄ b  + c⁄ d  = (ad + bc)⁄ bd.

Standards for Mathematical Practice:

Practice Standard 3: Construct viable arguments and critique the reasoning of others.

  • Are both of these representations correct? How do you know?
  • Do you agree or disagree with this solution? Defend your claim.
  • Prove that this number is equal to this expression. (e.g., 1 2⁄ 3  = 5 × 1⁄ 3 )

Practice Standard 4: Model with mathematics.

  • How can you use a number line to help you solve a fraction addition or subtraction problem with unlike denominators?
  • How do jumps on a number line represent addition? Subtraction?

Fraction Schema:

On a number line the unit whole is defined by the distance from 0 to 1. The number line can then be successively partitioned into units of the same distance. All numbers on the number line are positioned relative to that distance. The distance between units can then be partitioned repeatedly into any number of equal measures. Successive partitioning on the number line can lead to understanding of equivalent fractions as numbers that occupy the same point on the number line. Addition and subtraction can be shown on a number line as successive jumps in the same direction (addition) or a jump in the opposite direction (subtraction). Finding the location of an operation with jumps of different units (e.g. halves and thirds) is facilitated by partitioning each unit into an equivalent unit, e.g. halves are partitioned by three sections and thirds by two sections resulting in both having sixths as their unit.

Anticipated Student Responses: How might a student approach the task?

Task 1: ST Math JiJi Cycle Select Basket puzzles

Students will readily see how the fraction circles move along the number line and each new piece begins where the other pieces ended and moves right from that point on the number line.

  1. Students are able to use jumps from left to right to represent the action of the fraction circles moving along the number line to represent addition.
  2. Students see a whole circle and a 1⁄ 3  circle. They see the whole circle as 3⁄ 3  plus the 1⁄ 3  makes 4⁄ 3. They jump 4 jumps of size 1⁄ 3  to place the basket.

Task 2: Build a number line and locate 5⁄ 4  on your number line. Locate the point on your number line 4⁄ 3  to the right of 5⁄ 4. (or 4⁄ 3  and 3⁄ 2)

  1. Partitioning into 12ths: Students draw a number line and represent 5⁄ 4  as a point on the number line with no partitions marked. They partition the number line based on that point into 12ths and extend their number line 16⁄ 12  from 15⁄ 12. Watch for students who just plot the final answer on the number line without using it to show the action of adding and subtracting the fractions.
  2. Partitioning 4ths into 12ths: Students draw a number line, partition it into 4ths and represent 5/4 as a point on the number line with 5 jumps of 1⁄ 4. They partition each 4th into 3 equal intervals to make 12ths. They continue the number line 16⁄ 12.
  3. Two number lines: Students build one number line to show 4ths and one number line to show thirds using the same unit whole. They copy the size of the 4⁄ 3  onto the first number line.

What possible errors or misconceptions might students show?

Errors:
  • Students make calculation errors.
  • Students miscount points on the number line.
Misconceptions:
  • Students add the numerators and denominators.
  • Students plot 5⁄ 4  and 4⁄ 3  on the number line.
  • Students plot 5⁄ 4  and make four more jumps of random size.
  • Students use an algorithm to calculate the total but are not able to articulate it on the number line. (algorithmic understanding without conceptual foundation)

What questions will help students focus on the math in the ST Math Puzzle?

  • What do you see in this puzzle? <ask students to pair share at least 3 things they notice>
  • Where would you place the basket? How did you determine this?
  • How does the animation show the act of placing the numbers on the number line and adding them together or taking them away?
  • How could you represent the action of adding this number on the number line with a jump in place of a rolled out line?
  • What equation would represent this puzzle?
  • What multiplication equation could you write for this puzzle?
  • What is another name we could use for the blue pieces in this puzzle? (e.g., 3⁄ 2, 1 1⁄ 2, 6⁄ 4)

Select and Sequence:

What student strategies will support the lesson’s math goals?

The sequence below is one example. Your students may not have used all the strategies described below. Your students will vary in the language they use.

What sequence of strategies do you intend to use?

  1. Show an example of a misconception such as students who added 4 jumps to 5⁄ 4  without partitioning the number line to make accurate jumps. Have students identify the misconceptions.
  2. Students create a number line and show 5⁄ 4  as a jump on the number line. They rename 5⁄ 4  as 15⁄ 12. They partition the number line to the right of 5⁄ 4  into twelfths. They count over 16⁄ 12  Discuss what each jump/move represents in the problem.
  3. Have one or two students share how they solved the problem and represented it on the number line. Discuss and compare the jumps on the number line to the equations.
    • “I made my number line in fourths. Then I made 5 jumps of 1⁄ 4  to plot 5⁄ 4. I knew I had to make twelfths to add thirds and fourths so I divided the fourths into 3 equal parts and that made twelfths. 5⁄ 4  was the same as 15⁄ 12. I counted over 16⁄ 12  which is 4⁄ 3  to the right. I landed on 21⁄ 12, because 15 + 16 = 21. ”

Summary:

Whole Class Discussion

Teacher’s Role:

Highlight students’ strategies and reasoning that connect to the mathematics in the task.

How will you promote students’ strategies, representations and thinking?

Use student work and discuss ways to compare representations, record on chart paper or use a document camera to show student work.

Questions for the “5⁄4  + 4⁄3” that support learning goals

  • How are the number lines the same?
  • How are the number lines different?
  • Which jump on this student’s number line represents this equation? What equation does these jumps represent?
  • Is this student’s work clear without them explaining it to you? Explain.

Identify what you want students to take away from the discussion.

  • Students understand that if a number is at the same location as another number on a number line, the two numbers are equivalent.
  • Students understand that a point on a number line can have multiple names, yet represent the same number.
  • Students make connections between the jumps on a number line and expressions and equations. Connect back to the JiJi puzzles.
    • When I jump to the right 3 jumps of 1⁄4, it is the same as 31⁄4 .
  • Students understand how the number line jumps represent addition
  • Students understand why they need to find a common denominator for fractions in order to add or subtract them.
  • Students partition a number line several times in order to find a measure that allows them to name a point as accurately as possible. Then they connect this idea to the concept of creating common denominators.
